# Courses

A course is the main content unit in Konstantly. This guide covers how to manage courses through the API.

## The Course Object

<ResponseField name="id" type="integer">
    Unique identifier for the course
</ResponseField>

<ResponseField name="name" type="string">
    Name of the course
</ResponseField>

<ResponseField name="annotation" type="string">
    Brief description or summary of the course
</ResponseField>

<ResponseField name="isDraft" type="boolean">
    Whether the course is in draft state
</ResponseField>

<ResponseField name="createdAt" type="integer">
    Timestamp when the course was created
</ResponseField>

<ResponseField name="updatedAt" type="integer">
    Timestamp when the course was last updated
</ResponseField>

<ResponseField name="publishedAt" type="integer">
    Timestamp when the course was published
</ResponseField>

<ResponseField name="image" type="object">
    Course cover image information
    <Expandable title="properties">
        <ResponseField name="id" type="integer">
            Unique identifier for the image
        </ResponseField>
        <ResponseField name="original" type="string">
            URL to the full-size image
        </ResponseField>
        <ResponseField name="mini" type="string">
            URL to 36x36px thumbnail
        </ResponseField>
        <ResponseField name="small" type="string">
            URL to 330x130px image
        </ResponseField>
    </Expandable>
</ResponseField>

## List All Courses

<CodeGroup>
    ```bash Request
    curl -X GET "https://YOURSITE.konstant.ly/openapi/v1/courses" \
    -H "X-API-KEY: YOUR_API_KEY"
    ```

    ```json Response
    {
        "courses": [
    {
        "id": 123,
        "name": "Course name",
        "annotation": "Course annotation",
        "isDraft": false,
        "createdAt": 1507807711,
        "updatedAt": 1507808372,
        "publishedAt": 1508225229
    }
        ],
        "totalCount": 1
    }
    ```
</CodeGroup>

## Course Assignments

### Assign Course

<CodeGroup>
    ```bash Request
    curl -X POST "https://YOURSITE.konstant.ly/openapi/v1/courses/123/assignments" \
    -H "X-API-KEY: YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{
    "users": [1, 2, 3],
    "groups": [4, 5],
    "assignAt": 1634567890,
    "deadlineAt": 1644567890
}'
    ```

    ```json Response
    {
        "users": [...],
        "groups": [...]
    }
    ```
</CodeGroup>
